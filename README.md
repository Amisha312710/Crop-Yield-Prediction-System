# Crop-Yield-Prediction-System

Machine learning-based crop yield prediction system developed to analyze agricultural production patterns and improve yield forecasting using baseline and hyperparameter-tuned models.

---

## Overview

This project focuses on predicting crop yield using machine learning techniques and comparative model evaluation. Multiple models were trained and evaluated to improve prediction accuracy and optimize agricultural forecasting performance.

The project includes:
- Baseline machine learning models
- Hyperparameter-tuned models
- Data preprocessing and feature engineering
- Model evaluation and comparison
